# splunk_otel_flutter_session_replay_platform_interface Example

This example demonstrates the platform interface for Splunk session replay.

## Overview

This is a platform interface package that defines the API contract for session replay functionality. Platform-specific implementations provide the actual recording and playback capabilities.

## Note

The session replay module is currently under development. This example will be expanded as features are implemented.

## Usage

This package is not meant to be used directly. Instead, use the `splunk_otel_flutter_session_replay` package which implements this interface for Android and iOS platforms.

## Running the Example

```bash
cd example
dart main.dart
```
